また、コントリビューターは、当該コントリビューションを上記条件で提供するために必要な権利を有していること、第三者の権利を侵害していないこと、および所属組織・雇用主・委託元その他の第三者の許諾が必要な場合には必要な許諾を得ていることを表明するものとします。

Pull Request を提出する際は、Pull Request テンプレート内のコントリビューション同意欄にチェックしてください。同意が確認できない Pull Request は、原則としてレビューまたはマージできません。

## 開発の進め方

```bash
npm install
cp .env.example .env        # OPENAI_API_KEY / LLM_MODEL / EMBEDDING_MODEL を設定
npm run dev                 # PGliteで起動（http://localhost:3000）
npm run check               # typecheck + lint + test（提出前に必須）
```

- `npm run check` を通してから提出してください
